# Galería Modal

Este proyecto es una aplicación web de galería de imágenes simple y responsiva. Presenta una cuadrícula de imágenes que, al hacer clic en ellas, se abren en una vista modal. El modal permite a los usuarios navegar a través de las imágenes utilizando botones de "anterior" y "siguiente".

## Características

- **Diseño Responsivo**: Se adapta a diferentes tamaños de pantalla.
- **Vista Modal**: Las imágenes se amplían en un modal superpuesto al hacer clic.
- **Navegación**: Botones para recorrer las imágenes dentro del modal.
- **Estética Moderna**: Fondo con gradiente animado y efectos de hover en las imágenes.

## Tecnologías Utilizadas

Este proyecto fue construido utilizando tecnologías web estándar sin dependencias externas:

- **HTML5**: Estructura semántica de la página.
- **CSS3**: Estilos, diseño con Flexbox y Grid, animaciones y efectos visuales.
- **JavaScript (Vanilla)**: Lógica para la interactividad del modal y la navegación de imágenes.

## Cómo Usar

1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en tu navegador web preferido.
3. Haz clic en cualquier imagen para abrir el modal.
4. Usa los botones "anterior" y "siguiente" para ver otras imágenes.
5. Haz clic en la "X" o fuera de la imagen para cerrar el modal.
